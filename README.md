## üöÄ Getting Started

Follow these steps to get a local copy up and running on your machine.

### Prerequisites

Ensure you have the following installed on your system:
* **Node.js** (Version 18 or higher recommended)
* **NPM** (Included with Node.js)

### Installation Guide

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    cd your-repo-name
    ```

2.  **Install Dependencies**
    Install the required packages using NPM to ensure all modules are up to date.
    ```bash
    npm install
    ```

3.  **Environment Configuration**
    Security is paramount. You must configure your API keys locally.
    * Create or locate the `.env.local` file in the root directory.
    * Add your Gemini API key as shown below:

    ```env
    GEMINI_API_KEY=your_actual_api_key_here
    ```

4.  **Launch the Application**
    Start the development server:
    ```bash
    npm run dev
    ```

    You can now view the application running locally at `http://localhost:3000` (or the port specified in your terminal).
